# 获取用户信息

## 接口地址

```/api/v1/users/```

## 请求方法

```POST ```

## 接口变量
| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| user_ids | array    | yes      | 用户id   |
