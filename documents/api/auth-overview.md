# Overview - ThinkSNS plus
用户认证概述

## 自动认证+接口逻辑
```text
             登录 <--------------------------┓
              |                              |
              |                              |
              ↓                              |
         获取双 Token                         |
              |                              |
              |                              |
              ↓                              |
           请求 API <----------------------┓ |
              |                            | |
              |                            | |
              |                            | |
     ┌--------┻--------┐                   | |
     ↓                 ↓                   | |
  公开 API         非公开 API                | |
     |                 |                    | |
     |        ┌--------┻--------┐           | |
     |        ↓                 ↓           | |
     |     认证成功           认证失败         | |
     |        |                 |            | |
     |        |                 |            | |
     ↓        |                 ↓            | |
  处理请求 <---┛          请求刷新 Token 接口   | |
     |                          |             | |
     |                 ┌--------┻--------┐    | |
     |                 ↓                 ↓    | |
  完成处理            刷新失败           刷新成功 | |
                       |                 |    | |
                       |                 └----┛ |
                       └------------------------┛
```
